## 🎨 Color Scheme

- **Primary Orange**: #ff6b35 (Buttons, logo, accents)
- **Secondary Orange**: #f7931e (Gradients)
- **Text Dark**: #333 (Main text)
- **Text Light**: #666 (Descriptions)
- **Background**: #f5f5f5 (Light gray)
- **White**: #fff (Cards, inputs)

---

## 🔘 All Clickable Elements

### Home Page Navbar
- [Home] - Scroll to top
- [Why Us] - Scroll to features
- [Popular Items] - Scroll to popular section
- [Contact] - Scroll to footer
- [Order Now] - Go to menu page

### Throughout App
- All "Order Now" buttons → Menu page
- All "Add to Cart" buttons → Add item to cart
- [+] / [-] buttons → Change quantity
- [✕] button → Remove from cart
- [Home] button → Return to home
- [Continue Shopping] → Back to menu
- [Place Order] → Submit order
